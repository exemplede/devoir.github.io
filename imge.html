Galerie des Éléments Électroniques

Résistance

Une résistance est un composant électronique qui limite le flux de courant dans un circuit.

Condensateur

Un condensateur est un composant qui stocke l'énergie électrique dans un champ électrique.

Transistor

Un transistor est un composant utilisé pour amplifier ou commuter des signaux électroniques.

Oscilloscope

Un oscilloscope est un instrument utilisé pour visualiser des signaux électriques variables dans le temps.

Résistance Variable

Une résistance variable est un composant qui permet d'ajuster le flux de courant dans un circuit.

Condensateur Céramique

Un condensateur céramique est utilisé pour ses propriétés de haute fréquence et de faible inductance.

Transistor MOSFET

Un transistor MOSFET est utilisé pour ses capacités de commutation rapide et d'amplification efficace.

Oscilloscope Numérique

Un oscilloscope numérique permet de capturer et d'analyser des signaux électriques avec une haute précision.
